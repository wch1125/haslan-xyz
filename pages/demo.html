Feature: Sidenotes

Marginal Notes (Desktop Only)

Sidenotes appear in the margin when you hover over themLike this! Sidenotes let you add additional context without interrupting the main text flow. They're perfect for citations, tangential thoughts, or clarifications.. They're inspired by Edward Tufte's design principles.

Feature: Collapsible Sections

Hide/Show Content

Collapsible sections let you hide detailed information that not everyone needs:

Technical Implementation Details

The collapsible sections use the HTML5 <details> and <summary> elements, styled with CSS. JavaScript adds state persistence via localStorage.

Benefits:

  • Works without JavaScript (graceful degradation)
  • Remembers open/closed state between page loads
  • Accessible by default
  • Easy to style with CSS
Why Use Collapsible Sections?

Collapsible sections serve several purposes:

  1. Progressive disclosure: Show overview first, details on demand
  2. Reduced cognitive load: Don't overwhelm readers with everything at once
  3. Better scanning: Readers can quickly see section topics
  4. Mobile friendly: Reduces scrolling on small screens

Defined terms show preview popups when you hover over them (desktop only). Try hovering over these terms:

Feature: Dark Mode

Theme Toggle

Click the button in the sidebar or press Alt+D to toggle between light and dark modes. Your preference is saved in localStorage and persists between visits.

The dark mode uses CSS variables to smoothly transition all colors. No flash of unstyled content.
