Colophon

Abstract: How this site is built and what inspired it.

This is a static site with no build step, no CMS, and no framework. Just HTML, CSS, and vanilla JavaScript. Everything is version-controlled in Git and hosted on GitHub Pages.

I wanted something I could understand completely and fix when it breaks. The web has a long memory; platforms don't.

Design Inspirations

  • Gwern.net — The gold standard for personal wikis. Inspired the overall structure, the use of sidenotes, and the "evergreen" content philosophy.
  • Tufte CSS — The marginal notes pattern and the idea that typography can do heavy lifting.
  • C.S. Lewis's Narnia — The name and the subtle lion motif (gold accents, the loading animation, console easter eggs).

On the Name

H. Aslan is a refraction of my actual surname—"Haslun" with one letter changed. I wanted something that could mark the work without attaching my full identity to every essay, and the proximity to the Narnia lion was sitting right there, unavoidable once noticed.

The tagline matters more than the name. "Not a tame lion" is a line spoken about Aslan in the books—a warning from those who know him to those who don't. He's good, they say, but not safe; he won't be predicted or domesticated into comfortable categories. That seemed like a reasonable aspiration for a writing project.Whether the writing achieves this is a separate question.

The risk is grandiosity. What I hope prevents that is the texture of the essays themselves, which are tentative more often than declarative. The name borrows from a figure that knows things; the writing mostly doesn't.

Stack

  • HTML/CSS/JS — No frameworks.
  • Git — Version control for everything.
  • GitHub Pages — Hosting.

Total dependencies: zero. The site will work as long as browsers render HTML.

Typography

  • Body: System serif (Iowan Old Style → Palatino → Georgia)
  • UI: System sans-serif
  • Code: SF Mono → Consolas → Monaco

No web fonts. Fast and reliable.

Color

Monochrome with one accent: lion's mane gold (#B8860B). It appears in dropcaps, hover states, the loading animation, and the progress bar.

Steal This

If you want to make something similar, go ahead. The patterns aren't original. Personal websites are collaborative even when we don't talk to each other.

Starting from scratch?
  • Neocities — Free hosting, no coding required
  • GitHub Pages — Free hosting with Git
  • Bear Blog — Simple blogging

A single HTML page with your name and a paragraph is a personal website. Everything else is iteration.
